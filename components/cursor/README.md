# Cursor Component

## Назначение

Компонент `cursor` представляет набор результатов запроса, предоставляя интерфейс для итерации по результатам запросов (как документов, так и табличных данных).

## Основные файлы

- `cursor.hpp/cpp` - Основная реализация курсора

## Ключевые классы и функциональность

### Основные классы

- **`cursor_t`** - Главный класс курсора с поддержкой:
  - Результатов на основе документов
  - Результатов на основе таблиц (колоночное хранилище)

- **`error_t`** - Представление ошибок с кодами и сообщениями

- **`operation_status_t`** - Статус успеха/неудачи операции

### Коды ошибок

- `database_already_exists` - база данных уже существует
- `collection_not_exists` - коллекция не существует
- `sql_parse_error` - ошибка парсинга SQL
- И другие коды ошибок для различных ситуаций

### Методы навигации

- `has_next()` - проверка наличия следующего элемента
- `next_document()` - переход к следующему документу
- `get_document()` - получение текущего документа
- Методы для работы с табличными данными

## Использование

Курсор возвращается в результате выполнения запросов SELECT, INSERT, UPDATE, DELETE и других операций. Он предоставляет унифицированный интерфейс для доступа к результатам независимо от типа хранилища (документное или колоночное).
