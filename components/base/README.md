# Base Component

## Назначение

Компонент `base` предоставляет базовые структуры данных, используемые во всей системе для идентификации объектов базы данных.

## Основные файлы

- `collection_full_name.hpp` - Определение структуры полного имени коллекции

## Ключевые классы и функциональность

- **`collection_full_name_t`** - Структура, представляющая полное квалифицированное имя коллекции, включающее:
  - `unique_identifier` - уникальный идентификатор
  - `database` - имя базы данных
  - `schema` - имя схемы
  - `collection` - имя коллекции

- **`collection_name_hash`** - Функция хеширования для имен коллекций

- **Алиасы типов**:
  - `uid_name_t` - уникальный идентификатор
  - `database_name_t` - имя базы данных
  - `schema_name_t` - имя схемы
  - `collection_name_t` - имя коллекции

## Использование

Этот компонент является фундаментальным для всей системы и используется для однозначной идентификации коллекций и других объектов базы данных в распределенной среде.
