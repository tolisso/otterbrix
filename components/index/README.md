# Index Component

## Назначение

Компонент `index` предоставляет инфраструктуру индексирования для быстрого поиска и извлечения данных.

## Основные файлы

- `index.hpp/cpp` - Базовый класс индекса
- `index_engine.hpp/cpp` - Движок управления индексами
- `single_field_index.hpp/cpp` - Индекс по одному полю
- `hash_index.hpp/cpp` - Хеш-индекс
- `forward.hpp` - Предварительные объявления

## Ключевые классы и функциональность

### Основные классы

- **`index_t`** - Базовый класс индекса с:
  - Поддержкой итераторов
  - Операциями вставки и удаления
  - Операциями поиска

- **`index_engine_t`** - Движок управления индексами:
  - Создание индексов
  - Удаление индексов
  - Управление жизненным циклом

- **`single_field_index`** - Индекс по одному полю документа/колонки

- **`hash_index`** - Реализация хеш-индекса для быстрого поиска по равенству

### Основные операции

- **insert** - добавление записи в индекс
- **remove** - удаление записи из индекса
- **find** - поиск точного совпадения
- **lower_bound** - поиск первого элемента не меньше заданного
- **upper_bound** - поиск первого элемента больше заданного

### Типы данных

- **`index_value_t`** - Связывает ID документов или индексы строк с индексированными значениями

### Поддержка хранилища

- Индексы в памяти (memory-based)
- Индексы на диске (disk-based)

## Использование

Индексы используются для ускорения операций поиска, соединения и фильтрации данных. Они автоматически обновляются при вставке, обновлении и удалении данных.
