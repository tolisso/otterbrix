# OID Component

## Назначение

Компонент `oid` реализует генерацию и манипуляцию Object ID (идентификаторов объектов), аналогично MongoDB ObjectId.

## Основные файлы

- `oid.hpp` - Реализация OID

## Ключевые классы и функциональность

### Основной класс

- **`oid_t<T>`** - Шаблонный класс OID с компонентами:
  - **timestamp** (4 байта) - временная метка создания
  - **random value** (5 байт) - случайное значение
  - **increment** (3 байта) - инкрементальный счетчик

### Основные возможности

- **Генерация уникальных ID** - создание глобально уникальных идентификаторов
- **Преобразование в строку** - конвертация в шестнадцатеричное представление
- **Валидация** - проверка корректности OID
- **Операции сравнения** - полный набор операторов сравнения (<, >, ==, !=, <=, >=)
- **Специальные значения**:
  - Null OID - пустой идентификатор
  - Max OID - максимальное значение

### Поддержка хеширования

- **Hash function** - для использования в хеш-таблицах и unordered контейнерах

## Использование

OID используется для:
- Уникальной идентификации документов
- Уникальной идентификации объектов базы данных
- Временного упорядочивания (благодаря timestamp компоненте)
- Распределенной генерации ID без координации

```cpp
// Пример использования
auto doc_id = oid_t<>::generate();
std::string id_str = doc_id.to_string();
```
