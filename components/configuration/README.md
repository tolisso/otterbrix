# Configuration Component

## Назначение

Компонент `configuration` предоставляет централизованное управление конфигурацией системы базы данных.

## Основные файлы

- `configuration.hpp` - Определения конфигурации системы

## Ключевые классы и функциональность

- **`config`** - Главный класс конфигурации, содержащий все настройки системы

- **`config_log`** - Конфигурация логирования:
  - Путь к файлу логов
  - Уровень логирования (trace, debug, info, warn, error, critical)

- **`config_wal`** - Конфигурация Write-Ahead Log (журнала предзаписи):
  - Путь к WAL
  - Включение/отключение WAL
  - Синхронизация записей

- **`config_disk`** - Конфигурация дискового хранилища:
  - Путь к хранилищу
  - Включение/отключение дисковой персистентности

## Использование

Этот компонент используется для инициализации и управления всеми настройками системы при запуске и во время работы базы данных.
